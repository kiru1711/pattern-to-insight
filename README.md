**https://pattern-to-insight.vercel.app/**

**Problem**
-> Data Overload : Raw CSV data is difficult to interpret quickly
-> Manual Analysis : Requires coding or tools like Excel/Python
-> Lack of Visualization : Users struggle to identify trends, anomalies, and patterns
**Solution**
-> Automated Insights : Upload CSV → instant visual analysis
-> User-Friendly UI : No coding required for understanding data
-> Visual Representation : Charts for trends, comparisons, and distributions
-> Decision Support : Helps users quickly interpret data patterns
**Implementation**
-> Frontend : Built using React.js with Chart.js for interactive visualizations
-> Backend : FastAPI handles CSV upload and processing
-> Data Processing : Pandas and NumPy used for statistical analysis
-> API Flow : CSV → backend processing → JSON response → frontend charts
-> Features : Includes comparison, trend, distribution, anomaly, and threshold charts
